# 1&gt;搭建本地的 hexo 的博客

<a name="30zlk"></a>
## 1>官方文档  [https://github.com/hexojs/hexo](https://github.com/hexojs/hexo)


<a name="F5daQ"></a>
## 2>搭建博客
<a name="m1De6"></a>
### 1.全局安装 hexo，创建博客
```javascript
npm install hexo-cli -g
```
<a name="0h45k"></a>
### 2.初始化
```javascript
hexo init blog
```
<a name="sfiHV"></a>
### 3.启动本地 hexo-server
```javascript
hexo server
```


<a name="GAJwJ"></a>
## 4.新建一个 hexo 博客，富文本 makedown 文档
```javascript
$ hexo new "Hello Hexo"
```


<a name="AhhSk"></a>
## 5.打包发布到 hexo
```
//打包发布 hexo
hexo generate


//重启服务(可以看到新增的文档)
hexo server

```
